Sistema IoT para Automatización del Riego en Cultivo de Cacao Basado en Sensado de Humedad del Suelo y Ambiente



Se diseñó e implementó un sistema de monitoreo y control automático de riego orientado a la optimización de las condiciones de cultivo del cacao, mediante el uso de tecnologías IoT. El sistema permite una gestión inteligente del riego en función de los niveles de humedad detectados en el suelo y en el ambiente.


Para el control del riego, se integró un sensor de humedad del suelo, el cual monitorea en tiempo real el contenido hídrico de la tierra. Cuando el nivel de humedad desciende por debajo de un umbral definido, el sistema activa automáticamente una bomba de agua para iniciar el riego. Una vez que el sensor detecta que la humedad ha alcanzado un nivel óptimo o excedido un umbral superior, la bomba se desactiva, evitando el exceso de agua y promoviendo el uso eficiente del recurso hídrico.


Adicionalmente, se incorporó un sensor de humedad ambiental con el objetivo de monitorear las condiciones climáticas locales y asegurar un entorno propicio para el desarrollo del cacao. Esta información puede ser utilizada para análisis posteriores o futuras mejoras del sistema mediante lógica predictiva.


La interfaz de usuario fue desarrollada utilizando la plataforma Blynk, que permite la visualización remota de los datos y el control manual del sistema desde un dispositivo móvil. El sistema está basado en microcontroladores compatibles con IoT, y se comunica a través de protocolos como Wi-Fi y MQTT, permitiendo una supervisión eficiente y en tiempo real. Este proyecto integra conceptos de agricultura de precisión, automatización, control en lazo cerrado, y plataformas de IoT, ofreciendo una solución escalable y adaptable a diferentes entornos agrícolas.
