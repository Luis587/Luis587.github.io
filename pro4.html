Elevador controlado por asistente de voz



Se diseñó e implementó un sistema elevador automatizado dentro de un entorno de coworking especializado en robótica, con el propósito de optimizar el uso del espacio físico. El elevador incorpora una plataforma móvil destinada al almacenamiento de robots, permitiendo su resguardo cuando no están en operación y su despliegue cuando se requiere su utilización.


El Sistema fue desarrollado con un enfoque funcional y espacial, permitiendo que la plataforma descienda automáticamente para facilitar el acceso a los robots y se eleve cuando no están en uso, liberando así el área para otras actividades dentro del coworking. Esta solución maximiza la eficiencia del espacio disponible y contribuye a una organización operativa más flexible.


Como parte del desarrollo del sistema de elevador, se implementó un módulo de control por voz (ALEXA) que permite accionar el mecanismo de elevación mediante comandos verbales. Este asistente de voz fue acoplado al sistema de automatización para gestionar las órdenes de ascenso y descenso de la plataforma, proporcionando una interfaz de usuario intuitiva y manos libres.


Durante el desarrollo del sistema de control por voz del elevador, se implementó una arquitectura basada en Amazon Alexa como interfaz principal de interacción. La integración se logró mediante Node-RED, plataforma que permitió establecer flujos de comunicación entre los dispositivos compatibles con Alexa y el sistema embebido.


El procesamiento de los comandos de voz se canalizó a través de Node-RED, el cual actúa como puente entre el ecosistema de Alexa y el protocolo MQTT. Este protocolo se utilizó para la transmisión eficiente de mensajes hacia una ESP32, la cual opera como unidad de control en tiempo real.


La ESP32 recibe los comandos MQTT y ejecuta las acciones correspondientes para la activación o desactivación de los dos motores del elevador, controlando de manera precisa los movimientos de ascenso y descenso. Esta arquitectura distribuye adecuadamente las funciones entre los distintos componentes, permitiendo un control confiable, escalable y de bajo consumo energético.
