Simulación y control de un elevador industrial.



El objetivo principal de este proyecto fue diseñar y programar un sistema de control para un elevador automatizado, evaluando su comportamiento y precisión al detenerse en cada nivel del recorrido. La lógica de control fue desarrollada utilizando TIA Portal (Totally Integrated Automation), implementando un autómata programable (PLC) para gestionar las secuencias de movimiento, las condiciones de parada y la interacción con los sensores de piso.


La simulación y validación del sistema se realizaron mediante la integración con Factory I/O, una plataforma de simulación industrial 3D que permitió verificar el comportamiento dinámico del elevador en un entorno virtual. Esta comunicación entre TIA Portal y Factory I/O se estableció mediante una conexión OPC o driver compatible, permitiendo ejecutar pruebas en tiempo real y garantizar el funcionamiento adecuado del sistema antes de su implementación física.


Este proyecto demuestra el uso de herramientas de automatización industrial modernas para el desarrollo, prueba y simulación de sistemas mecatrónicos bajo condiciones controladas.
